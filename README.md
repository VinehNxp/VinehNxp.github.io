# 💈 ZapVitrine — Barbearia Prime

Landing page moderna, responsiva e focada em conversão para uma barbearia fictícia chamada **Barbearia Prime**.

Este projeto faz parte da proposta **ZapVitrine**, um modelo de criação de vitrines digitais rápidas para pequenos negócios locais, com foco em agendamento pelo WhatsApp, apresentação profissional e presença online simples.

---

## 🔥 Visão geral

A **Barbearia Prime** é uma landing page desenvolvida para demonstrar como pequenos negócios podem ter uma presença digital mais profissional sem depender apenas de redes sociais.

O site apresenta:

* Página inicial com chamada de impacto
* Serviços e preços
* Galeria visual
* Avaliações de clientes
* Chamada direta para WhatsApp
* Informações de contato e localização
* Layout responsivo para celular, tablet e desktop

---

## 🎯 Objetivo do projeto

O objetivo principal é criar uma página comercial simples, moderna e vendável para negócios locais.

Este projeto pode ser usado como demonstração para vender sites parecidos para:

* Barbearias
* Salões de beleza
* Manicures
* Estéticas
* Lojas locais
* Prestadores de serviço
* Pequenos negócios que vendem pelo WhatsApp

## 💬 Integração com WhatsApp

Os botões do site direcionam o usuário para o WhatsApp com mensagens pré-definidas, facilitando o agendamento.

Exemplo:

```txt
Olá! Gostaria de agendar um horário na Barbearia Prime.
```

Para alterar o número do WhatsApp, edite a constante no arquivo:

```js
const WHATSAPP_NUMBER = "5599999999999";
```

## 🛠️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/VinehNxp/zapvitrine-barbearia-prime.git
```

Entre na pasta do projeto:

```bash
cd zapvitrine-barbearia-prime
```

Abra o arquivo:

```txt
index.html
```

Ou use a extensão **Live Server** no VS Code.
